# Zenith

Premium static web app skeleton built for speed and design.

## Tech Stack

- **Runtime:** Bun
- **Framework:** React 19 + TypeScript
- **Build:** Vite 7
- **Styling:** Tailwind CSS v4
- **Deploy:** GitHub Actions (Pages)

## Quick Start

```bash
bun install
bun start    # Dev server at http://localhost:5173
bun run build # Production build
```

## Deployment

Push to `main`. GitHub Actions automatically deploys to [checkout here](https://kohld.github.io/zenith/).
